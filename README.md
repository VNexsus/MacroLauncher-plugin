## Плагин замены шрифтов в презентациях в редакторах Р7-Офис
Плагин позволяет создавать кнопки в верхней панели для вызова макросов в документе.

<img alt="Preview" width="590px" src="https://github.com/VNexsus/MacroLauncher-plugin/blob/main/Preview.png">


## Функциональные возможности:
Выбор имеющихся документе макросов и добавление отдельной кнопки для его вызова на верхнюю панель редактора. Кнопки с выбранными макросами отображаются для каждого документа отдельно. При необходимости лишние кнопки можно удалить.

## Применимо для продуктов:
*	Десктоп-редакторы Р7-Офиc 2025.3.1.923 и выше

## Как установить
*	Для установки плагина скачайте файл <a href="https://github.com/VNexsus/MacroLauncher-plugin/blob/main/MacroLauncher.plugin">MacroLauncher.plugin</a> следуйте инструкциям на сайте <a href="https://support.r7-office.ru/desktop_editors/plugins-and-macros/plugins-plugins-and-macros/adding-plugins/p7-office-docs-on-premises/">Р7-Офиc<a/>
